# 🔐 Full Stack Authentication App

A complete **Authentication System** built with **Flutter (Frontend)** and **Node.js + MongoDB (Backend)**, featuring **JWT Auth** and **Firebase Social Login**.

---

## 🚀 Features

*  User Registration & Login (Email + Password)
*  Secure Authentication using JWT
*  Backend with Node.js & Express
*  MongoDB Database Integration
*  Firebase Social Login (Google Sign-In)
*  Clean Flutter UI
*  Form Validation & Error Handling
*  Forgot Password
*  Email Verification

---

## 🛠️ Tech Stack

###  Frontend

* Flutter
* Dart
* Provider (State Management)

###  Backend

* Node.js
* Express.js
* MongoDB (Mongoose)
* JWT Authentication
* Firebase 

### 🔐 Authentication

* Firebase Authentication (Google Sign-In)
